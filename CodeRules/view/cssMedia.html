css media---概述

通过媒体查询为不同的设备和大小配置不同的样式。

代码展示

  /* media */
  /* 横屏 */
  @media screen and (orientation:landscape){
  }
  /* 竖屏 */
  @media screen and (orientation:portrait){
  }
  @media only screen and (max-width:768px) {
	html { font-size: 204.8px }
